# for
``` js
for (let i = 0; i < 10; i++) {
	code...
}
```


# for … in
``` js
const sample = {
	name : 'sample1',
	year : 2003,
}

for (let unit in sample) {
	console.log(unit)
}

// name year group
```
- map을 for…in 하면 key값을 꺼내옴

``` js
const sample = ['sample1', 'sample2']

for (let unit in sample) {
	console.log(unit)
}

// 0 1
```
- list를 for…in하면 인덱스를 꺼내옴

# for … of
``` js
const sample = ['sample1', 'sample2']

for (let unit in sample) {
	console.log(unit)
}

// sample1 sample2
```
- list를 for…of 하면 값을 꺼내옴
