# 같다
## `==`
- 값만 비교
- 숫자 5와 문자열 5를 같다고 해버리는 대참사;;
- true와 1이 같다고 하는 대참사;;

## `===`
- 값과 타입을 같이 비교
- 비교의 정상화


# 논리연산자
## and `&&` or `||`
- 반환할때 true false 대신 두 비교값 중 truthy, falsy한 값을 반환함
- 이를 이용해서 단축평가(Short Circuit Evaluation) 가능

## 단축평가
### &&
- 좌측이 true → 우측 값 반환
- 좌측이 false → 평가할 필요도 없이 좌측값 반환

### ||
 - 좌측이 true → 평가할 필요도 없이 좌측값 반환
 - 좌측이 false → 우측 값 반환
