## 제네릭 함수

### 형태

```Swift
func 함수이름<T>(매개변수: T) -> T {
    // T 타입 데이터 처리
}
```

- T는 제네릭 타입 매개변수로 실행 시점에 결정 됨
- T는 정해진 규칙이 아닌 그저 관례임, 원하는 용어 사용 가능
