### Task

> [!important] 메인 쓰레드가 아닌 다른 쓰레드로 일을 넘기는 함수

```Swift
print("hello")
Task{
		var number = 0
		for i in 0...200000 {
				number += i
		}
		print("for \(number) times")
}
print("world")
```

```Swift
hello
world
for 20100 times
```
