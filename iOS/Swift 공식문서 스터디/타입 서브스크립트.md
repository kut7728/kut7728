# 타입 서브스크립트

## 정의

- 오버라이딩이 불가하면 static
- 가능하면 class로 선언해주고
- subscript 메서드로 조회 가능하도록 선언

```Swift
struct Stack {
    static var stack: [Int] = [0, 1, 2, 3]
 
    static subscript(index: Int) -> Int {
        return stack[index]
    }
}
```
